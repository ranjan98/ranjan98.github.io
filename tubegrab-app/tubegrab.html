About TubeGrab

TubeGrab is a sleek, dark-mode desktop app for downloading YouTube videos, audio, and entire playlists. It wraps the powerful yt-dlp engine in a friendly native GUI — paste a link, pick a quality, and go.

Key Features

  • Video downloads with quality caps (Best / 4K / 1080p / 720p / 480p), merged to mp4
  • Audio-only mode: best original format, MP3 320/192 kbps, or M4A
  • Subtitle downloads (.srt) — manual subs and auto-generated CC, any language
  • Playlist detection with numbered, foldered downloads
  • Live progress bar with download speed and a cancel button
  • Auto-retry and resume on flaky connections
  • Fully threaded — the UI never freezes mid-download
